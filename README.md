# springFarmEasy
소개 

귀농 인구를 위한 지역 및 농작물 추천 서비스	

작업 기간 : 1개월-2개월

작업 인원 : 2명자

스프링 프레임워크를 이용해 구현

메인 화면

프로그램을 시작하면 메인 화면이 나타나며 우측 상단의 버튼을 이용해 로그인 / 회원 가입 가능
